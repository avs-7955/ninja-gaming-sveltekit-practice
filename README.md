## Run Locally

Clone the project

```bash
  git https://github.com/avs-7955/polling-app-svelte.git
```

Go to the project directory

```bash
  cd polling-app-svelte
```

Install the packages

```bash
  npm i
```

Run the server

```bash
  npm run dev
```

Click the localhost link shown in the console.
